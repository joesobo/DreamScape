# Dream Scape - V 1.0

A small 2d platformer created in Unity

## Setup/Installing

Import all files into Unity Hub and restart

## Usage

Controls for Player

* a/d         - move left/right
* w           - jump
* f           - start wave
* space	      - shoot dream at infection (TODO)
* arrow keys  - attack in direction (TODO)
