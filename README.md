# Dream Scape V 1.0

A small 2d platformer created in Unity

## Setup/Installing

Import all files into Unity Hub and restart

## Usage

a/d         - move left/right
f           - start wave
space       - jump
right click - shoot at point
arrow keys  - attack in direction
